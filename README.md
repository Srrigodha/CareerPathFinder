# 🚀 AI-Powered Career Path Finder System

An intelligent web-based platform that helps students make informed career decisions using Artificial Intelligence. The system generates personalized career roadmaps, recommends exams and scholarships, and provides real-time guidance through an interactive chatbot.


## 📌 Overview

Choosing the right career path is challenging due to the overwhelming amount of information available today. This project solves that problem by providing a **personalized, AI-driven guidance system** that adapts to each student’s profile.

It analyzes user inputs such as academic stage, interests, and career goals to generate structured and actionable insights, helping students plan their future effectively.


## ✨ Key Features

- 🎯 **Personalized Career Roadmaps** using AI  
- 📚 **Exam Recommendations** with difficulty levels  
- 🎓 **Scholarship Tracking** with deadline alerts  
- 🤖 **AI Chatbot Assistant** for real-time guidance  
- 📊 **Progress Tracking System** with completion status  
- 📈 **Success Rate Prediction** for exams  
- 🌙 **Dark/Light Mode UI** for better user experience  


##  Tech Stack

**Frontend:**
- HTML5  
- CSS3  
- JavaScript (ES6+)  
- Chart.js  

**Backend:**
- Python (Flask)  

**Database:**
- SQLite  

**AI Integration:**
- Groq API (Llama-3.3-70B Model)  

##  System Architecture

The system follows a **three-tier architecture**:

- **Presentation Layer:** User Interface (HTML, CSS, JavaScript)  
- **Application Layer:** Backend Logic (Flask API)  
- **Data Layer:** SQLite Database + AI API Integration
